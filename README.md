# 客戶推薦頁產生器

一個純前端的菁菁房地產顧問工具。貼上物件網址後，可以整理標題、備註、聯絡資料，並產出可分享的客戶頁連結。

## 使用方式

建議用本機伺服器預覽，這樣短網址物件名稱與照片才會透過 `/api/property-meta` 正確抓取：

```bash
node server.js
```

再打開 `http://127.0.0.1:4173/`。

如果只直接開啟 `index.html` 或使用 `python3 -m http.server`，瀏覽器只能讀靜態檔，無法執行短網址解析 API。

## 功能

- 解析 `https://`、`www.`、`x.ychouse.tw/` 開頭的物件網址
- 自動過濾重複網址
- 透過 `/api/property-meta` 讀取物件頁標題、實際物件照片、總價、格局、建坪、車位、地址等重點
- 可替每個物件補客戶備註
- 客戶頁會依區域與社區產生快速分類，點選後可跳到相關物件段落
- 客戶點擊物件、喜歡、電話、LINE 時會記錄到後台統計
- 統計後台在 `/stats`，不用密碼；客戶頁不會顯示後台入口
- 預設使用菁菁資料：電話 `0910-692-946`、LINE `nicolyyin`、證號 `115登字第502348號`
- 自帶菁菁風格的工作台與客戶提案頁版型
- 顧問卡人像使用 `assets/jingjing-portrait.png`
- 產生 `#page=` 分享連結
- 可下載獨立 HTML 客戶頁
- 支援手機與桌面版面

## 物件資料擷取

客戶頁會使用短網址頁面解析出的標題與照片。若物件頁無法公開讀取、沒有提供圖片，或本機沒有啟動 `server.js`，產生分享頁按鈕會保持停用，避免出現圖文不符的物件卡。

## 統計後台

啟動 `node server.js` 後，打開 `http://127.0.0.1:4173/stats` 可查看統計。客戶頁不會顯示後台入口。點擊紀錄保存在 `.data/click-stats.json`，此資料夾已加入 `.gitignore`。
